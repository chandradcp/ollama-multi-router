Run this command in your terminal to test OpenAI-compatible chat completions:

curl -X POST http://localhost:20128/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer default-insecure-key" \
  -d '{
    "model": "llama3.1:8b",
    "messages": [{"role": "user", "content": "Hello! Why is Ollama Multi Router awesome?"}],
    "stream": true
  }'

Use the official Python `openai` SDK to connect directly to the router:

from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:20128/v1",
    api_key="default-insecure-key"
)

response = client.chat.completions.create(
    model="llama3.1:8b",
    messages=[{"role": "user", "content": "Explain load balancing in 2 sentences."}]
)

print(response.choices[0].message.content)

Use the official Node.js / TypeScript `openai` SDK:

import OpenAI from 'openai';

const openai = new OpenAI({
  baseURL: 'http://localhost:20128/v1',
  apiKey: 'default-insecure-key',
});

async function main() {
  const completion = await openai.chat.completions.create({
    messages: [{ role: 'user', content: 'What is high availability?' }],
    model: 'llama3.1:8b',
  });
  console.log(completion.choices[0].message.content);
}
main();
